TL;DR¶

This is an straightforward example of the vardef decorator.

See also

For better understanding of this decorator you can look at the other examples in the Usage Examples section. The Basic Usage Example is a good start!

tldr_example.py¶
from vardef import vardef


def run_example():
    @vardef
    def foo() -> int:
        return 42

    print(foo)
    # 42

    print(type(foo))
    # <class 'int'>


if __name__ == "__main__":
    run_example()
